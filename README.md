# yume-nikki-hotel-manager
A JavaFX-based Hotel Management and Guest Registry System featuring room management, automated billing, invoice generation, persistent storage, and a retro pixel-inspired graphical interface.

## Overview

Yume nikki is a JavaFX-based Hotel Management System designed to manage hotel operations including room registration, guest check-in/check-out, billing, invoice generation, and guest history tracking. The application features a retro pixel-art interface inspired by Yume Nikki and provides an interactive desktop experience.

## Features
Room Registration and Availability Tracking
Guest Check-In and Check-Out
Additional Services Management
Automatic Bill Calculation
Invoice Generation
Guest Search and Filtering
Persistent Data Storage
Sound Effects and Animated User Interface
Guest History and Status Monitoring
##Technologies Used
Java
JavaFX
FXML
